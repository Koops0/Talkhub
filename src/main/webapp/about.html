Project Overview

TalkHub is a web chat server built using Java Web Application project. It employs WebSockets along with HTML, CSS, and JavaScript. The core functionality includes:

  • Multiple chat rooms where users can send messages to others in the same room.
  • Dynamic room creation with random alphanumeric codes.
  • User authentication with usernames.
  • Real-time user greetings and notifications when new users join a room.
  • Timestamped chat messages.
  • Simple UI with the ability to join existing rooms, create new rooms, and send messages.
  • Features such as a list of current chat rooms and message sending via Enter key and "send" button.

The project aims for a user-friendly interface with room for additional interactions and improvements beyond the basic requirements.
